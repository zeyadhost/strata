# Strata

A browser-based singleplayer mining game built with Phaser 3.

## Requirements

- Node.js 20+
- npm 10+

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Starts the Vite dev server.

| Service | URL |
|---------|-----|
| Game    | http://localhost:5173 |

## Build

```bash
npm run build
```

Output goes to `dist/`.
